Logic in an Illogical Sense

Rough Explanation

I was browsing Wikipedia and happened upon an article about Logic Gates, and after reading for a bit I thought it might be fun to try to make an adder inside the game Minecraft. So I learned the basics of the logic gates and using Truth Tables I made a ripple-carry adder to calculate addition up to ten. Below will be some pictures of the thing with captions. Sidenote: I got Dunning-Krugered about halfway into building this project. This happened after I found out that I actually know almost nothing about boolean logic other than the super basic aspects.
